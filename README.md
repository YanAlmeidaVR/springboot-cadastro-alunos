Projeto: Sistema de Cadastro de Alunos
Objetivo do Projeto

Desenvolver uma aplicação Java + Spring Boot para gerenciar alunos, aplicando boas práticas de arquitetura, camadas bem definidas, validação, DTOs e documentação de API.

Funcionalidades (Escopo Inicial)
Aluno

Cadastrar aluno

Listar alunos

Buscar aluno por ID

Atualizar dados

Remover aluno

Ativar / desativar aluno (soft delete)

Tecnologias Utilizadas

Java 17+

Spring Boot

Spring Web

Spring Data JPA

Hibernate

H2 (dev) / PostgreSQL (prod)

Bean Validation

Swagger (OpenAPI)

Lombok (opcional)
